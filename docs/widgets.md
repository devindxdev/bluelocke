# Widgets

Bluelocke supports both lock screen and home screen widgets.

## Lock Screen Widgets

- `Inline` text style
- `Circular` icon style
- `Rectangular` compact detail style

## Home Screen Widgets

- `Small` widget
- `Medium` widget

## Refresh Behavior

Widgets use configurable poll periods to balance freshness and battery usage.

- API poll period
- Remote poll period
- Charging remote poll period
- Night-time variants for each period

You can tune these under app settings -> advanced widget settings.
