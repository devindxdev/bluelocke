# Control Center

You can launch Bluelocke actions quickly from iOS Control Center using Shortcuts.

## Setup

1. Create individual Shortcuts for frequent commands (for example `lock`, `auto lock`, `status`).
2. Add each Shortcut to Control Center from iOS customization settings.
3. Tap once to execute the command through Scriptable.

## Tips

- Keep one Shortcut per action for fast access.
- Use clear Shortcut names so Siri and Control Center labels are unambiguous.
