# Installation

## Requirements

- iPhone or iPad with the Scriptable app installed
- A GitHub release containing the `bluelocke.js` build asset

## Steps

1. Install the Scriptable app from the App Store.
2. Open this repository releases page:
   - `https://github.com/devindxdev/bluelocke/releases`
3. Download/copy the latest `bluelocke.js` build content.
4. In Scriptable, create a new script named `bluelocke`.
5. Paste the script content and save.
6. Run it once and complete setup (region, brand, credentials, PIN, preferences).
7. Add widgets and/or Shortcuts.

## Upgrading

Use the in-app `Info` screen and install prompt, or replace script content manually with a newer release build.
