# Installation

Bluelocke runs inside the Scriptable iOS app.

## Simple 3-Step Flow

1. Install Scriptable from the App Store: [Install Scriptable](https://apps.apple.com/us/app/scriptable/id1405459188?uo=4)
2. Download the latest script file directly: [Download bluelocke.js](https://github.com/devindxdev/bluelocke/releases/latest/download/bluelocke.js)
3. In the Files app, move `bluelocke.js` into `iCloud Drive/Scriptable`

![GitHub release page showing the bluelocke.js file in release assets](./images/install-release-download.png)

## Then Open It

Open Scriptable and tap `bluelocke` to run setup.
