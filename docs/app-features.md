# App Features

The app screen is designed for quick common actions and status visibility.

## Main Actions

- Refresh status
- Lock / unlock
- Climate on / off
- Charge start / stop (for supported vehicles)
- Charge limit selection (for supported vehicles)
- Open car location in Maps

## Configuration

- Region, manufacturer, credentials, and PIN
- Temperature unit and distance unit
- Seat climate defaults
- Widget appearance and update prompt behavior
- Custom climate profiles
- Charge limit profiles

## Visuals

- Vehicle image with built-in overrides for known models/trims/colors
- Last update time
- Odometer and 12V indicators where available
